#랜딩페이지

## 구현의도
홈페이지와는 다른 랜딩페이지를 만들면서, 사람들의 시선이 분산되는 것을 방지하고자 하였고, 최대한 상품에 집중할 수 있게 만들었다. 첫페이지에 제품의 사진을 보여주면서 페이지의 성질을 알려줄 수 있는 슬라이드를 첨부한 것은 그러한 의도로 넣게되었다.
